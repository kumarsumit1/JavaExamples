Which of the following is true about DTR garbage collection?


Options are :

  • DTR garbage collection automatically delete unused image layers, thus saving you disk space. (Correct)
  • DTR Garbage collection removes DTR images that are expired.
  • DTR garbage collection removes unreferenced image layers from DTR's backend storage. (Correct)
  • DTR Garbage collection removes duplicated images.

Answer :DTR garbage collection automatically delete unused image layers, thus saving you disk space. DTR garbage collection removes unreferenced image layers from DTR's backend storage.
