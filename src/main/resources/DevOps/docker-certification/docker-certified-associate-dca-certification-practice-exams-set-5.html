  1.  REPOSITORY        TAG            IMAGE ID             CREATED                  SIZE
  2.  img-1            latest        fd484f19954f        23 seconds ago      7 B (virtual 4.964 MB)
  3.  img-2            latest        fd484f19954f        23 seconds ago      7 B (virtual 4.964 MB)
  4.  img-3            latest        fd484f19954f        23 seconds ago      7 B (virtual 4.964 MB)
  5.  img-4            latest        ba458d17824c        30 seconds ago      7 B (virtual 4.964 MB)

  • docker image prune --all

Answer : docker rmi -f fd484f19954f
